第 1 关

认识三色

Go GC 用三种颜色标记堆对象：
⬜ 白色：还未被 GC 访问到
◑ 灰色：已被发现，但子节点还没全部扫描
⬛ 黑色：已扫描完毕，子节点也都处理了

第 3 关

写屏障触发

Go 1.14+ 的混合写屏障规则：
• 栈对象默认视为黑色，无需写屏障
• 堆对象被新引用时，被引用对象变灰
• 堆对象原有引用被删除时，被删除的对象变灰
