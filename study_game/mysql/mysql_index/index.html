第 5 关

索引选型

表：orders(id, user_id, status, amount, created_at)
最高频的 3 种查询如下：

-- 查询1（最高频） SELECT id, amount FROM orders WHERE user_id=? AND status=?;
-- 查询2 SELECT * FROM orders WHERE user_id=? ORDER BY created_at DESC;
-- 查询3 SELECT COUNT(*) FROM orders WHERE status=?;

以下 4 种索引方案，哪个最优？

A　INDEX(user_id)，INDEX(status)，INDEX(created_at)
B　INDEX(user_id, status, amount)，INDEX(user_id, created_at)
C　INDEX(status, user_id)，INDEX(created_at)
D　主键 id 已经够了，不需要额外索引
