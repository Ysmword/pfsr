第 4 关

Next-Key Lock

同样的表（id: 3, 7, 12, 20），RR 隔离级别：

SELECT * FROM t WHERE id = 7 FOR UPDATE; -- id=7 存在，RR 隔离级别

id=7 存在，这条查询加的 Next-Key Lock 是？

提示：Next-Key Lock = 间隙锁 + 记录锁，是一个左开右闭区间。

A. (3, 7] — 左边间隙 + id=7 记录锁（临键锁）
B. (7, 12) — 右边间隙锁
C. (3, 12) — 整个周围范围都锁
D. 只加 id=7 记录锁，不加间隙锁
